# Phone Automator landing page

This is a simple static landing page for GitHub Pages.

## How to publish it on GitHub Pages

1. Create a new public repository on GitHub.
2. Put `index.html` in the repository root.
3. Push the files.
4. In repository settings, open **Pages**.
5. Set the source to your main branch and root folder.
6. Wait until GitHub publishes the site.
7. Copy the site URL, for example `https://yourname.github.io/repo-name/`.

## What to change before using it

- Replace `your@email.com` in `index.html`
- Add your real product price
- Add your crypto payment link or invoice link
- Update the page text with your brand name

## After publishing

Paste the GitHub Pages URL into the NOWPayments website field.
